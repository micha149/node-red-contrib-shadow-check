Checks whether the sun currently shines into a configured window.

Inputs

payload.altitude number
The altitude angle of the sun in degrees.
payload.azimuth number
The azimuth angle of the sun in degrees. North is 0 degrees.
payload.sunInSky boolean
If set to false, all geometry checks are skipped and the output payload is false.

Outputs

payload boolean
true if the sun shines into the configured window, otherwise false.
topic string
The configured output topic. Defaults to sunInWindow.

Details

The input format is compatible with sun position nodes such as Sun Position.

All window and overhang measurements must use the same unit. The node only compares relative geometry, so centimeters, inches or any other unit can be used consistently.

Azimuth
The azimuth angle of the window in degrees. A north-facing window is 0, a south-facing window is 180.
Width (a)
The width of the window.
Height (b)
The height of the window.
Inset (c)
The offset between the wall face and the actual window.
Overhang Offset Left (d)
The distance from the left side of the window to the left edge of the roof.
Overhang Offset Right (e)
The distance from the right side of the window to the right edge of the roof.
Overhang Offset Top (f)
The vertical distance from the top edge of the window to the bottom edge of the roof. For sloping roofs, measure the vertical distance to a projected line, not to the actual roof edge.
Overhang (g)
The horizontal distance from the wall face to the roof edge.
